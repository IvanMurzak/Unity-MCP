# Support

For bugs, setup problems, or packaging issues, open an issue in the main repository:

- [Unity-MCP Issues](https://github.com/IvanMurzak/Unity-MCP/issues)

When reporting a VS Code extension problem, include:

- VS Code version
- extension version
- OS version
- whether the workspace was trusted
- whether the project already had the Unity MCP package installed
- the relevant `Unity MCP` output channel logs

If the problem involves Unity launching, also include:

- Unity editor version
- whether the issue happened on plain launch or MCP-connected launch
- whether it was the first Unity open after installing the package

If Unity closes unexpectedly during setup, reopen the project and retry once. If the issue repeats, capture Unity `Editor.log` and file a bug with the exact step that triggered it.
